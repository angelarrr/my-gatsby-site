# Angela's Gatsby Site

This site was made using [Gatsby](https://github.com/gatsbyjs/gatsby).
It consists of a homepage with a header and YouTube video component.

Please follow these instructions to get this project running on your local machine.

```shell
# Clone repository
$ git clone https://github.com/angelarrr/my-gatsby-site

# Go into directory
$ cd my-gatsby-site
```

Install dependencies using NPM or Yarn:

Using NPM:
```shell
# Install dependencies
$ npm install

# Start development server
$ npm run develop
```

Using Yarn:
```shell
# Install dependencies
$ yarn

# Start development server
$ yarn develop
```

Once server has started, go to [http://localhost:8000/](http://localhost:8000/)
